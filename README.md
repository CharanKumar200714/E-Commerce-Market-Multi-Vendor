🚀 Project Overview
A comprehensive Full Stack Multi-Vendor E-Commerce Marketplace engineered to allow multiple independent sellers to list, manage, and sell their products on a single platform. The system utilizes a robust relational database architecture to handle multi-tenant user authentication (Admin, Vendor, Customer), distinct product inventories, shopping carts, and order processing workflows.

✨Project scope (Features)
Multi-Tenant Authentication: Separate dashboards, permissions, and registration flows for Admins, Vendors, and Customers.Vendor Storefront Management: Dedicated portals for vendors to execute full CRUD operations on their own product catalogs, images, and prices.Global Marketplace Feed: Seamless customer browsing interface with product searching, categorization, and advanced filtering options.Unified Checkout & Cart: Handles real-time price calculations, stock verification, and centralized order placement for multiple vendor products.Vendor Order Tracking: Automatic order partitioning allowing individual vendors to update fulfillment status for their specific items.

🛠️ Tech Stack
Frontend: React, Tailwind CSS, JavaScript (ES6)
Backend: Node.js, Express.js
Database: MySQL / PostgreSQL (SQL Relational Database)
ORM / Driver: Sequelize / mysql2
